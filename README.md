# Kropp Fitness

## English

### Description
A landing page for a fitness club built with HTML, CSS, and JavaScript. The page includes sections for training types, schedule, membership plans, and a contact map powered by Yandex Maps API.

### Technologies
- HTML5
- CSS3
- JavaScript
- Yandex Maps API

### Project Structure
```
kropp-fitness/
├── index.html       # Main page
├── styles.css       # Styles
├── script.js        # JavaScript
├── fonts/           # Fonts
├── icons/           # SVG icons
└── images/          # Images
```

### How to Run
Simply open `index.html` in your browser — no build tools required.

### Live Demo
